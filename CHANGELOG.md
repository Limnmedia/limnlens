# Changelog

## v0.1.0-alpha

Initial alpha release preparation.

### Added

- Browser-based two-point TPS scale calculation workflow.
- Image point picker for Point A and Point B.
- Effective focal length, angle of view, field of view, crop factor, pixel density, and related camera-matching results.
- Measurement confidence feedback.
- JSON calculation profile export.
- No-build local development workflow.
- Plain Node.js math regression tests.

### Known limitations

- Picker zoom/view modes are still basic.
- Distortion modeling is represented but not fitted or applied.
- JSON profile import is not implemented yet.
- Multi-sample lens maps are planned but not implemented.
- YAML calibration profile export is planned but not implemented.
- KaTeX currently loads from a CDN.
